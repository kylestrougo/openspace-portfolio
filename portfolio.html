OpenSpace Portfolio 2019

American Museum of Natural History - Hayden Planetarium

Github: Kyle Strougo

Refined a photogrammetry process for the American Museum of Natural History OpenSpace Project. OpenSpace is a NASA supported, open-source, interactive data visualization developed for planetary domes, flat screens, VR and networking for world organizations. Developed a Python program for automated digital terrain model generation utilizing data sets from NASAâ€™s Mars Reconnaissance Orbiter (MRO).

MRO global coverage includes data sets from CTX (8 meters per pixel) and HIRISE (6 meters per pixel) instuments. The process produces the necessary files for map projecting these digital terrain models into OpenSpace software. The program utilizes NASA's Ames Stereo Pipeline (ASP) and Integrated System for Imagers and Spectrometers (ISIS).
